# Units Converter

A simple Python project for converting units of measurement.

This project stores unit values relative to a base unit and uses them to convert between different units. It currently supports several categories, including time, length, mass, area, volume, speed, and temperature-related unit data.

## Features

- Convert between units of time
- Convert between units of length
- Convert between units of mass
- Convert between units of area
- Convert between units of volume
- Convert between units of speed
- Contains data for temperature units
- Supports many common, scientific, historical, and approximate units
- Written in pure Python
- No external dependencies required
